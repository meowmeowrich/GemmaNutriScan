# GemmaNutriScan

A React + Tailwind web app for nutritional analysis.

## Prerequisites

- [Node.js](https://nodejs.org/) installed
- [LM Studio](https://lmstudio.ai) installed - CORS enabled, serve on local network ON.
- [google/gemma-3-12b](https://huggingface.co/google/gemma-3-12b-it) installed

## Run Locally

1. Clone the repo:

`git clone https://github.com/meowmeowrich/GemmaNutriScan.git
cd GemmaNutriScan`

Change the IP to your own:
Inside `App.tsx`

Install dependencies:
`npm install`

Install Vite:
`npm create vite@latest`

Start the app:
`npm run dev`
